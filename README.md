> extremely generic game about zombies and mafia to demonstrate gly engine.

### Build with npm

```
npm install
```

```
npm run build
```

### Build with lua

```
wget get.gamely.com.br/cli.lua -o cli.lua
```

```
lua cli.lua build-html src/game.lua --fengari --atob
```
